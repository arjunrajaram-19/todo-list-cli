# To-Do List Manager (CLI)

A command-line based task manager built using Python.  
This application allows users to manage daily tasks efficiently with support for due dates and persistent storage.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as complete
- Clear all tasks
- Persistent storage using file handling
- Due date support with validation

## Example Usage

1. Add a task  
2. Assign a due date  
3. Mark it complete  
4. Edit or delete as needed  

All data is saved automatically in a file.

## Tech Used
- Python
- File Handling

## How to Run

1. Download or clone the repository  
2. Open terminal in the project folder  
3. Run:

python main.py
